Smart Home System

Project Overview

The Smart Home System is an IoT-enabled home automation solution. This system features a wireless sensor network based on the ESP32 microcontroller. It integrates cloud connectivity via MQTT and Firebase for real-time monitoring and control through a React Native mobile application. The project focuses on creating an accessible, user-friendly home automation interface.

Key Features

  • Wireless sensor network using ESP32 microcontrollers
  • Cloud integration with MQTT for sensor data transmission
  • Real-time monitoring and control capabilities via Firebase
  • Mobile app control using React Native framework
  • User authentication and access control features
  • Scalable architecture for adding new sensors and devices

Technologies Used

  • Microcontroller: ESP32
  • Communication Protocols: MQTT
  • Cloud Platform: Firebase
  • Mobile Development: React Native

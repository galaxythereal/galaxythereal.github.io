Automotive Gateway Module

Project Overview

The Automotive Gateway Module is a multi-protocol communication interface designed for modern vehicles. This module supports CAN, LIN, and Ethernet communication, enabling seamless data exchange between various vehicle systems and external diagnostic tools. It adheres to AUTOSAR standards for software architecture and interoperability. The gateway is built on the STM32H7 microcontroller platform utilizing FreeRTOS for real-time task management.

Key Features

  • AUTOSAR-compliant software architecture
  • Real-time message routing with minimal latency
  • Supports CAN-FD protocol for high-speed communication
  • Ethernet TCP/IP stack for diagnostic over IP (DoIP)
  • LIN bus interface for local network communication
  • Diagnostic data logging capabilities for fault analysis

Technologies Used

  • Microcontroller: STM32H7 Series
  • Real-Time OS: FreeRTOS
  • Communication Protocols: CAN-FD, LIN, Ethernet (TCP/IP)
  • Standards: AUTOSAR
